# gallery-drive-viewer

Nâng cấp trang `/gallery/{token}` từ "nút mở Google Drive" thành viewer ảnh nhúng (grid masonry + lightbox + tải từng ảnh + tải toàn bộ qua Drive), đọc ảnh trực tiếp từ `drive_url` sẵn có. Port ý tưởng UI/UX từ module-album-anh (React Native + Supabase) sang stack web hiện tại (React + Vite + Express/MySQL), bỏ hết chức năng thừa (bình luận, yêu thích, NAS, mật khẩu folder, my-albums, ZIP client-side).
